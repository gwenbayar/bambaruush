# Flows — Narrative Manifest

> This is the ordered manifest of every story-flow file in Bambaruush: scenes, chapters, and lesson scripts in the sequence the child experiences them. The **Story Guide** (`../references/story-guide.md`) holds the *rules*; this folder holds the *moments* that obey them.
>
> **Read this file first** when adding or reviewing a flow, to find the correct narrative position and the next file number.

## How flows are named
`NN-short-slug.md` — zero-padded number for narrative order, then a short kebab-case slug.
Examples: `01-opening-scene.md`, `02-sar-naadam-race.md`, `10-tsagaan-sar-feast.md`.

Leave gaps in numbering if helpful (e.g., jump 01 → 05) so scenes can be inserted later without renumbering everything.

## Continuity rules for flows
- Each flow must clear every **hard invariant** in the Story Guide (§2, §7, §8).
- Each flow's lesson(s) must map to a **fixed mini-game frame** (§6) owned by the correct character.
- A season's flows must match what **§5** says that season teaches and unlocks.
- Sky-stars and Odon must be awarded consistently across flows; Od's growth only moves forward.
- If a new flow seems to require changing a rule, surface it and update the Story Guide deliberately — never let canon drift silently.
